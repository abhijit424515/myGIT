# Blender 

### Shortcuts
| Key 1   | Use                    | Extras    |
| ------- | ---------------------- | --------- |
| n       | Toggle top right panel |           |
| g       | grab                   | g + x/y/z |
| alt + g | reset transformation   |           |
| s       | scale                  | s + x/y/z |
| alt + s | reset scaling          |           |
| r       | rotate                 | r + x/y/z |
| alt + r | reset rotation         |           |
|         |                        |           |

| Key 2                      | Use                                     | Extras |
| -------------------------- | --------------------------------------- | ------ |
| ctrl + a -> All Transforms | Set the current g/s/r values as default |        |
| shift + a                  | Add menu                                |        |

---

- New objects will appear at the location of the 3D cursor
